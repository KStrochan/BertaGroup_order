# Berta HoReCa — сайт онлайн-замовлень

Готовий B2B-каталог без реєстрації клієнтів. Замовлення надсилаються одним Telegram-ботом у групу **Berta Order**. Сайт показує повідомлення **«Замовлення відправлено»** тільки після успішної відповіді Telegram API.

## Що вже реалізовано

- 969 товарів з наданих Excel-прайсів:
  - Food — 515;
  - NONFood — 454.
- Початкові ціни з Excel, без застосування знижок.
- Пошук і фільтрація за категоріями.
- Кошик зі зміною кількості та автоматичним розрахунком суми.
- Збереження кошика у браузері клієнта.
- Оформлення без реєстрації:
  - ФОП / компанія;
  - контактна особа;
  - телефон;
  - місто;
  - вулиця;
  - номер будинку;
  - приміщення, дата доставки та коментар — необов’язкові.
- Доставка позначена як доступна по Львівській області.
- Сервер повторно перевіряє ціни за власним каталогом — клієнт не може підмінити суму в браузері.
- Захист від простого спаму та обмеження частоти запитів.
- Якщо замовлення довге, бот надсилає його кількома послідовними повідомленнями.
- База даних відсутня: історія замовлень залишається тільки в Telegram-групі.

## Запуск на комп’ютері

Потрібен **Node.js 20 або новіший**.

### 1. Створіть файл `.env`

Скопіюйте `.env.example` у `.env`:

```bash
cp .env.example .env
```

У Windows можна просто створити копію файлу та перейменувати її на `.env`.

### 2. Додайте актуальний токен

Відкрийте `.env` і вставте новий токен бота:

```env
TELEGRAM_BOT_TOKEN=ВАШ_АКТУАЛЬНИЙ_ТОКЕН
TELEGRAM_CHAT_ID=-1004299599812
PORT=3000
```

Токен не можна публікувати або завантажувати в GitHub. Файл `.env` уже доданий до `.gitignore`.

### 3. Запустіть сайт

```bash
npm start
```

Відкрийте:

```text
http://localhost:3000
```

## Перевірка налаштування

У браузері відкрийте:

```text
http://localhost:3000/api/health
```

Очікувана відповідь після додавання токена:

```json
{
  "ok": true,
  "telegramConfigured": true,
  "productCount": 969
}
```

Після цього додайте тестовий товар у кошик і оформіть замовлення. Повідомлення **«Замовлення відправлено»** з’явиться тільки після того, як Telegram підтвердить надсилання в групу.

## Важливо перед публікацією

- Використовуйте тільки новий токен, який не надсилали у відкритих повідомленнях.
- На хостингу додайте `TELEGRAM_BOT_TOKEN` як секретну змінну середовища.
- `TELEGRAM_CHAT_ID` уже встановлено: `-1004299599812`.
- Бот повинен залишатися учасником групи **Berta Order** і мати право надсилати повідомлення.
